Regras dos contrutores:

1. Toda classe possui um contrustor.
2. O construtor é um método com o mesmo nome da classe.
3. Se nenhum construtor for encontrado na classe,
o compilador inclui um construtor padrao.
( atribui os valores default para os atributos da 
classe).
4. Se colocarmos pelo menos 1 construtror na classe,
este sera usado, e o compilador nao interfere.
5. O construtor é um metodo sem valor de retorno
definido.
6.O construtor é executado uma unica ez, e quando
executado, é pelo operador new.
